# Гэп-анализ: AI для авто-карго Китай→Россия

**Дата:** 2026-03-19

## Рынок AI для логистики: что существует и работает

### Запад (США/UK/Израиль)

**Wisor.ai** (Израиль)
- Что делает: AI-котировки для грузоперевозчиков. Полное КП за 60 сек. Динамическое ценообразование — ML подбирает оптимальную цену на основе истории побед/проигрышей по каждому клиенту и маршруту.
- Принцип: «умная надстройка поверх существующих систем» — не заменяет TMS, а сидит сверху.
- Результаты: время на КП **–85%**, конверсия растёт, прогнозирование ставок.
- Не подходит нам: не работает с Россией, не для авто-карго, не для Telegram, английский язык.

**Raft.ai** (UK, $30M Series B)
- Что делает: AI-автоматизация операций. Обрабатывает документы из email, создаёт записи в TMS автоматически.
- Результаты: 35% инвойсов без участия человека, 40% — одно касание. **2 часа экономии на каждую отправку.** 300 000 отправок/мес через платформу.
- Не подходит нам: не для России, не для фурного карго, email вместо Telegram.

**Sedna** (UK)
- Что делает: AI-коммуникации для логистики. Автосортировка, маршрутизация, обогащение данными.
- Результаты: объём email **–85%**. Интеграция с Shipamax для автоматической обработки документов.
- Не подходит нам: рынок карго Китай→Россия живёт в Telegram, не email.

**Project44** (США)
- Что делает: real-time visibility, предиктивные ETA.
- Не подходит нам: под санкциями, не работает с Россией.

### Китай

**易境通 Yijingtong** — SaaS для карго, включая маршрут Китай→Россия
- Что делает: полная система управления грузоперевозками. API-парсинг товаров с 1688/Taobao. Умная консолидация. Мультиязычный, мультивалютный. Дашборды, отчёты.
- Результаты: **1 000+ компаний.** 89% снижают операционные расходы на **30–45% за 3 месяца**. Удержание клиентов **+41%**, средний чек **+28%**.
- Не подходит нам напрямую: тяжёлая система (TMS/WMS), требует полной миграции, не AI — классическая автоматизация, дорого и сложно для мелких карго на 20–50 человек.

**小蓝本 Xiaolanben / 探迹 Tungee** — AI-лидогенерация для грузоперевозчиков
- Что делает: ищет потенциальных клиентов по данным маркетплейсов, определяет ЛПР, даёт контакты.
- Результаты: стоимость привлечения с $3 000 до **$1 200 (–60%)**. Качественных лидов **+250%**. Время КП с 48ч до 2ч.
- Не подходит нам напрямую: заточен под китайский рынок, не для русскоязычных компаний.

## Наше позиционирование

**«Wisor для авто-карго Китай→Россия»**

Не заменяем то, как компания работает. Не заставляем мигрировать в новую систему. Садимся поверх Telegram и Excel как AI-слой, который делает существующие процессы быстрее.

## Что строим (реальное, без фантастики)

### MVP (2–4 недели)

**AI-расчёт по ссылке на товар в Telegram**
- Клиент кидает ссылку на 1688 + количество
- Бот парсит страницу: вес, габариты, цена (как 易境通 через API)
- Считает по ставкам ЭТОЙ компании: авто/ЖД/авиа (как Wisor для ocean)
- Выдаёт полное КП в Telegram за 30 сек
- Провалидировано: Wisor –85% время, 易境通 +28% средний чек

**Автостатусы клиентам**
- Менеджер отмечает этап фуры: выехала / граница / таможня / Москва / выдача
- Всем 20–50 клиентам этой фуры — автоматическое уведомление в Telegram
- Провалидировано: Sedna –85% коммуникационного шума

### v2 (месяц 2–3)

**AI-ассистент в Telegram**
- Отвечает на «где мой груз?» автоматически (по данным из статусов)
- Отвечает на FAQ: сроки, маршруты, документы, условия
- Переводит рус↔кит для двуязычных компаний
- Провалидировано: азиатская логистическая сеть — 85% запросов решает бот, расходы на сервис –55%

### v3 (месяц 4–6, когда есть данные)

**Динамическое ценообразование**
- ML на истории: какие ставки выигрывают клиентов, какие теряют
- Рекомендация оптимальной цены для конкретного клиента/маршрута
- Провалидировано: Wisor — ML на win/loss history
