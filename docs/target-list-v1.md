## 0. QUICK START: ручной outreach без инфраструктуры

> **Принцип:** Не ждём автоматизацию. Первые 10-15 контактов делаем руками.
> Это валидирует messaging до того, как потратим деньги на инфраструктуру.

### Шаг 1: Поиск селлеров на Авито (30 мин)

Открыть Авито в браузере и выполнить поисковые запросы ниже. Для каждого запроса — выбрать 2-3 продавцов по критериям (много товаров, каталожные фото, ключевые слова «из Китая» / «опт» / «от производителя»).

**Горячие запросы (города-хабы, высокий score):**

| # | Запрос на Авито | Город | Категория | Искать |
|---|----------------|-------|-----------|--------|
| 1 | `одежда оптом из Китая` | Владивосток | Одежда | Магазины с >10 позиций |
| 2 | `женская одежда от производителя` | Владивосток | Одежда | Каталожные фото, 1688-стиль |
| 3 | `наушники оптом` | Новосибирск | Электроника | Прямые поставки |
| 4 | `смартфон из Китая` | Новосибирск | Электроника | Новые аккаунты, много позиций |
| 5 | `одежда 1688` | Москва | Одежда | Прямое упоминание 1688 = hot lead |
| 6 | `товары для дома опт из Китая` | Москва | Дом/красота | Крупные магазины |
| 7 | `автоаксессуары из Китая` | Екатеринбург | Авто | Оптовики |
| 8 | `детская одежда из Китая` | Москва | Детские | Фабричные фото |
| 9 | `косметика из Китая оптом` | Владивосток | Красота | Прямые импортёры |
| 10 | `платья опт Китай` | Новосибирск | Одежда | Много размеров/цветов |

**Фильтры Авито при поиске:**
- Тип объявления: «Компания» (отсекает частников)
- Сортировка: «по дате» (свежие = активные продавцы)
- Смотреть профиль продавца: количество объявлений, дата регистрации, отзывы

### Шаг 2: Оценка каждого продавца (2 мин на продавца)

Для каждого найденного продавца записать:

| Поле | Как проверить |
|------|---------------|
| Имя продавца | Никнейм на Авито |
| URL профиля | Ссылка на профиль (не на объявление) |
| Кол-во позиций | Указано в профиле |
| Дата регистрации | В профиле, раздел «На Авито с...» |
| Ключевые слова в заголовках | «из Китая», «опт», «от производителя», «1688» |
| Каталожные фото | Студийные/фабричные фото = прямой импортёр |
| Город | Указан в профиле |
| Примерный score | Посчитать вручную по таблице в разделе 1 |

### Шаг 3: Написать первые сообщения (5 мин на сообщение)

- **Score 80+** → Шаблон E (партнёрский подход из AICA-46)
- **Score 60-79** → Шаблоны A-D (выбрать по контексту)
- Отправить через встроенную систему сообщений Авито
- **Важно:** НЕ отправлять ссылки, НЕ упоминать бота — мы карго-компания

### Результат Quick Start

10 сообщений отправлены за 1-2 часа. Если >1 ответ — messaging работает, масштабируем. Если 0 ответов — корректируем шаблоны до покупки инфраструктуры.

| # | Критерий | Вес | Сигнал |
|---|----------|-----|--------|

## 5. Персонализация для Top-10 (Hot leads)

Для каждого hot lead (score 80+) готовим индивидуальный подход:

### Фреймворк персонализации

Для каждой компании из top-10 фиксируем:

1. **Товарный фокус** — что именно продаёт (конкретные SKU, категории)
2. **Объём** — количество позиций, частота обновлений
3. **Ценовой сегмент** — ниже медианы = прямой импорт, подтверждение ICP
4. **Гео-сигнал** — город-хаб или нет, что это значит для логистики
5. **Hook для первого сообщения** — конкретный товар для варианта E
6. **Follow-up стратегия** — какую конкретную выгоду подчеркнуть

### Шаблон персонализированной заметки

```
## Компания #{N}: {имя_продавца}
- Профиль: {url}
- Score: {score}/100 ({breakdown})
- Товарный фокус: {конкретные товары}
- Объём: {N} позиций, обновления каждые {N} дней
- Ценовой сегмент: на {N}% ниже медианы в {категория}
- Город: {город} — {значение для логистики}

### Первое сообщение (вариант E — адаптированный):
> Добрый день! Работаем с логистикой Китай-Россия, заметил ваш профиль —
> {items_count} позиций в {категория}, впечатляет. Видел «{конкретный_товар}» —
> мы как раз возим подобное из Гуанчжоу. Возможно, будем полезны друг другу.
> Интересно пообщаться?

### Follow-up фокус:
- Упор на: {конкретная выгода — скорость расчёта / отслеживание / цена}
- Social proof: «работаем с {N} продавцами в {категория}»
```

## 7. Рыночный контекст (март 2026)

### Тренды Авито × Китай

- **200K+ товаров** с доставкой из Китая на Авито (кросс-бордер запущен конец 2024, масштабируется)
- **Рост заказов одежды** из Китая на Авито: **x7.4** за окт 2025 → фев 2026
- **Рост электроники** из Китая: **x3.6** за тот же период
- **100K+ китайских селлеров** на Ozon (~20% всех продавцов), удвоение за год на Ozon+WB
- **Топ-категории** с долей китайских продавцов: электроника (~69%), товары для дома (28-31%), одежда (28%), автозапчасти (+65% за год)
- **Средний срок доставки:** 2-3 недели (Авито Доставка + Почта России)
- **Безопасная сделка:** заморозка средств до получения — снижает барьер

### Ключевые инсайты для outreach

1. **Селлеры во Владивостоке** — ближе всего к Китаю, часто сами ездят за товаром. Максимальная плотность прямых импортёров.
2. **Новосибирск** — перевалочный хаб, оптовые склады. Продавцы масштабнее, больше позиций.
3. **Москва** — максимальный объём, но больше посредников. Фильтровать по import_keywords.
4. **Одежда** — самый массовый сегмент импорта, рост x7.4 создаёт волну новых импортёров. Выделяться через скорость расчёта.
5. **Электроника** — ~69% продавцов из Китая, выше средний чек. Выделяться через автоматизацию.
6. **Новые селлеры** — удвоение китайских продавцов за год означает поток новичков, которым нужна логистика. Искать аккаунты <1 года.
