### Критерии отбора (из scoring-модели)

| # | Критерий | Вес | Сигнал |
|---|----------|-----|--------|
| 1 | Import keywords в заголовке | +20 | «опт», «из Китая», «от производителя», «1688», «прямая поставка» |
| 2 | Много товаров (>10 позиций) | +20 | Серьёзный продавец, постоянный импорт |
| 3 | Каталожные фото (>=5) | +15 | Студийные/фабричные фото = товар с завода |
| 4 | Новый аккаунт (<1 года) | +15 | Новые импортёры, ещё не нашли стабильную логистику |
| 5 | Цена ниже медианы (на 30%+) | +15 | Прямой Китай, без посредников |
| 6 | Город-хаб (Владивосток, Новосибирск) | +10 | Основные точки входа китайских товаров |
| 7 | Мало позиций (<=3) | +5 | Свежий продавец, может быть новый импортёр |

## 5. Персонализация для Top-10 (Hot leads)

Для каждого hot lead (score 80+) готовим индивидуальный подход:

### Фреймворк персонализации

Для каждой компании из top-10 фиксируем:

1. **Товарный фокус** — что именно продаёт (конкретные SKU, категории)
2. **Объём** — количество позиций, частота обновлений
3. **Ценовой сегмент** — ниже медианы = прямой импорт, подтверждение ICP
4. **Гео-сигнал** — город-хаб или нет, что это значит для логистики
5. **Hook для первого сообщения** — конкретный товар для варианта E
6. **Follow-up стратегия** — какую конкретную выгоду подчеркнуть

### Шаблон персонализированной заметки

```
## Компания #{N}: {имя_продавца}
- Профиль: {url}
- Score: {score}/100 ({breakdown})
- Товарный фокус: {конкретные товары}
- Объём: {N} позиций, обновления каждые {N} дней
- Ценовой сегмент: на {N}% ниже медианы в {категория}
- Город: {город} — {значение для логистики}

### Первое сообщение (вариант E — адаптированный):
> Добрый день! Работаем с логистикой Китай-Россия, заметил ваш профиль —
> {items_count} позиций в {категория}, впечатляет. Видел «{конкретный_товар}» —
> мы как раз возим подобное из Гуанчжоу. Возможно, будем полезны друг другу.
> Интересно пообщаться?

### Follow-up фокус:
- Упор на: {конкретная выгода — скорость расчёта / отслеживание / цена}
- Social proof: «работаем с {N} продавцами в {категория}»
```

```bash
python -m scripts.avito_outreach.scoring --threshold 40
```

### Шаг 3: Отбор top-50

```sql
-- Hot leads (10 слотов)
SELECT * FROM scored_sellers
WHERE is_candidate = 1 AND score >= 80
ORDER BY score DESC LIMIT 10;

-- Warm leads (30 слотов)
SELECT * FROM scored_sellers
WHERE is_candidate = 1 AND score BETWEEN 60 AND 79
ORDER BY score DESC LIMIT 30;

-- Cool leads (10 слотов, резерв)
SELECT * FROM scored_sellers
WHERE score BETWEEN 40 AND 59
ORDER BY score DESC LIMIT 10;
```

### Шаг 4: Заполнение таблицы

1. Скопировать данные из SQL-результатов в таблицу выше
2. Назначить шаблон сообщения по правилам:
   - Score 80+: вариант E (партнёрский)
   - Score 60-79, import_keywords: вариант C
   - Score 60-79, many_items: вариант B
   - Score 60-79, конкретная цена: вариант D
   - По умолчанию: вариант A
3. Подготовить персонализированные заметки для top-10

### Шаг 5: Интеграция с sender-ботом

## 7. Рыночный контекст (март 2026)

### Тренды Авито × Китай

- **200K+ товаров** с доставкой из Китая на Авито (запуск кросс-бордер в конце 2024)
- **Рост заказов одежды** из Китая на Авито: x7.4 за последние 5 месяцев
- **Рост электроники** из Китая: x3.6 за тот же период
- **Средний срок доставки:** 2-3 недели (Авито Доставка + Почта России)
- **Безопасная сделка:** заморозка средств до получения — снижает барьер

### Ключевые инсайты для outreach

1. **Селлеры во Владивостоке** — ближе всего к Китаю, часто сами ездят за товаром. Максимальная плотность прямых импортёров.
2. **Новосибирск** — перевалочный хаб, оптовые склады. Продавцы масштабнее, больше позиций.
3. **Москва** — максимальный объём, но больше посредников. Фильтровать по import_keywords.
4. **Одежда** — самый массовый сегмент импорта, но и самая высокая конкуренция. Выделяться через скорость расчёта.
5. **Электроника** — выше средний чек, продавцы технологически грамотнее. Выделяться через автоматизацию.
