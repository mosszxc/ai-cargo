## Поток B: Outreach на селлеров WB/Ozon (ОСНОВНОЙ)

### Контекст

- Цель: привлечь селлеров как клиентов нашего карго-сервиса
- По DECISIONS.md: **мы = карго-компания**, бот — наш инструмент
- Каналы: Авито, TG-чаты импортёров, WB/Ozon форумы, 1688-чаты
- Мы говорим: «Работаем с логистикой Китай→Россия»
- Reference Group: другие импортёры одежды/электроники из Китая

### Скрипт B1: Холодное сообщение — Авито (через чат Авито)

**Когда:** Нашли селлера с признаками импорта из Китая (score 60+)
**Канал:** Встроенные сообщения Авито
**Важно:** НЕ упоминать бота, НЕ отправлять ссылки — Авито банит за ссылки

```
Добрый день! Увидел ваш профиль — впечатляющий ассортимент
{категория: одежда / электроника / и т.д.}.

Мы занимаемся доставкой из Китая (Гуанчжоу, Иу, Шэньчжэнь).
Возим авто, авиа, ж/д — зависит от объёма и сроков.

Если работаете с китайскими поставщиками — возможно, будем полезны.
Откуда сейчас везёте товар?
```

**Почему работает:**
- Мы — карго, не SaaS (по DECISIONS.md)
- Никаких ссылок (Авито банит)
- Конкретные города (Гуанчжоу, Иу) — показывает экспертизу
- Вопрос в конце — открывает диалог

### Скрипт B2: Холодное сообщение — Telegram (из чата импортёров)

**Когда:** Селлер задаёт вопрос в TG-чате про доставку/карго
**Канал:** Ответ в чате или личное сообщение

```
{Имя}, работаем с логистикой из Китая. По {маршрут, который обсуждался} —
можем посчитать за пару минут, если скинете ссылку на товар.

Возим авто из Гуанчжоу (18-25 дней), авиа (5-7 дней).
Считаем быстро — у нас автоматизировано.
```

**Почему работает:**
- Контекстный ответ на реальный запрос
- «Считаем быстро — у нас автоматизировано» — намёк на скорость без прямой продажи
- Конкретные сроки доставки — язык карго-профессионала

### Скрипт B3: Переход из Авито в Telegram

**Когда:** Селлер ответил на Авито, диалог идёт
**Цель:** Перевести общение в Telegram (где работает бот)

```
Удобнее общаться в Telegram — там быстрее отвечаем
и можем сразу посчитать доставку.

Напишите @{наш_тг} — я {имя менеджера}, отвечу сам.
```

**Почему работает:**
- Естественная причина перехода (удобство)
- Личное имя — не спам, а человек
- В Telegram бот считает мгновенно — клиент увидит скорость первым сообщением

### Скрипт B4: Первый расчёт в Telegram

**Когда:** Селлер перешёл в Telegram
**Цель:** Показать скорость расчёта, зацепить

```
Привет, {имя}! Рад, что написали.

Чтобы посчитать доставку — просто скиньте ссылку на товар с 1688
и количество. Получите расчёт с разбивкой за минуту.

Или опишите товар текстом — тоже сработает.
```

**Логика:** Клиент кидает ссылку → бот мгновенно считает → клиент видит скорость → wow-эффект. Не объясняем что это бот. Клиент думает «у них просто быстрая система».

### Скрипт B5: Follow-up после расчёта (не заказал)

**Когда:** Клиент получил расчёт, но не оформил заказ

```
{Имя}, как вам расчёт? Если нужно пересчитать с другим маршрутом
или объёмом — скиньте новую ссылку, посчитаем.

Кстати, по {категория товара} сейчас {инсайт: ставки снизились /
маршрут через Казахстан дешевле / авиа актуально к сезону}.
```

**Почему работает:**
- Забота, не давление
- Предложение пересчёта = ещё один контакт
- Рыночный инсайт — показывает экспертизу

### Скрипт B6: Партнёрский подход (Hot lead, score 80+)

**Когда:** Крупный селлер с большим ассортиментом
**Канал:** Telegram, после перехода с Авито

```
{Имя}, посмотрел ваш каталог — {N} позиций в {категория}, солидно.

Мы возим из Китая и работаем с несколькими продавцами вашего масштаба.
Видел «{конкретный товар}» — как раз возим подобное из Гуанчжоу.

Можем предложить:
- Быстрый расчёт: кидаете ссылку — получаете стоимость за минуту
- Отслеживание: статус груза в реальном времени
- Выделенный менеджер

Интересно пообщаться подробнее?
```

**Почему работает:**
- Персонализация (конкретный товар из каталога)
- Reference Group Matching: «работаем с продавцами вашего масштаба»
- «Быстрый расчёт» — показывает скорость без прямого упоминания бота

### Скрипт B7: TG-чаты — холодный outreach

**Когда:** Пишем первыми людям из чатов WB/Ozon/1688, ещё не задававшим вопрос
**Канал:** Личное сообщение в Telegram

```
Добрый день! Увидел вас в {название чата}.

Мы карго-компания, возим из Китая (Гуанчжоу, Иу).
Работаем с продавцами WB/Ozon — считаем доставку быстро,
без форм и долгого ожидания.

Если везёте или планируете везти товар из Китая —
можем быстро посчитать конкретную позицию. Работаете с Китаем?
```

## Обработка возражений

### «Уже работаем с карго»

```
Отлично, что уже возите из Китая! А как быстро вам считают?
Можем посчитать тот же товар — сравните скорость и ставки.
Просто скиньте ссылку, бесплатно.
```

### «Дорого»

```
Понимаю. Давайте сравним конкретно: скиньте ссылку на товар,
который везёте сейчас — посчитаем и сравним со своим карго.
Если дороже — честно скажем.
```

### «Не верю, что AI считает точно»

```
Правильный вопрос. Проверьте сами: скиньте ссылку на 1688 —
получите расчёт за 30 секунд. Если цифры не сходятся с вашим
текущим карго — расскажите, разберёмся вместе.
```

### «Пришлите КП/прайс»

```
Конечно. Но прайс без контекста — это просто цифры.
Лучше так: скиньте одну ссылку на 1688 — я покажу,
как посчитаем именно ваш товар. 30 секунд.
После этого отправлю с привязкой к вашей специфике.
```

## Каналы поиска клиентов

1. **Авито** — профили с признаками импорта (одежда, электроника, товары для дома)
2. **TG-чаты** — «WB продавцы», «Ozon sellers», «1688 импортёры», «Карго Китай»
3. **WB/Ozon форумы** — вопросы про доставку и закупки
4. **Реферальный** — каждый довольный клиент → просьба порекомендовать

**Признаки горячего лида:**
- Задаёт вопрос про стоимость доставки из Китая
- Продаёт одежду/электронику/товары для дома
- Ссылается на 1688 или Alibaba
- Жалуется на медленный расчёт или высокие ставки текущего карго

---

## Правила outreach

1. **Никогда не спамить.** Максимум 3 касания на контакт (первое + 2 follow-up)
2. **Персонализировать.** Каждое сообщение содержит минимум 1 элемент, специфичный для получателя
3. **Не продавать в первом сообщении.** Цель — открыть диалог
4. **На Авито — никаких ссылок и ботов.** Мы карго-компания
5. **Фиксировать всё.** Каждый контакт → в CRM-трекер
6. **Учиться.** Каждую пятницу: что сработало, что нет, какие возражения новые
