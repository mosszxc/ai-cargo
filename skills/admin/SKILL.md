## Утилита rate_manager.py

Все операции через CLI. Путь: `/home/dev-moss/ai-cargo/skills/admin/rate_manager.py`

```bash
RM="python3 /home/dev-moss/ai-cargo/skills/admin/rate_manager.py --company test-company --caller-id <sender_telegram_id>"
```

**ВАЖНО:** Всегда передавай `--caller-id` с Telegram ID отправителя. Скрипт проверяет права доступа на уровне кода. Без `--caller-id` доступ будет разрешён (CLI-режим).

## Операции (ТОЛЬКО для ID 5093456686 и 291678304)

### Показать настройки (Курсы, Услуги)

Триггеры: "покажи настройки", "конфиг", "курсы валют", "доп услуги"

```bash
$RM show-config
```

Форматируй результат:
```
⚙️ **Настройки FastCargo**

💱 **Курсы валют:**
USD/CNY: 7.25
USD/RUB: 88.50

🛠 **Услуги:**
Страховка: 3%
Обрешётка: 40%
Паллетирование: 16%

📦 **Наценки на категории:**
Электроника: x1.5
Хрупкое: x1.2
```

### Обновить курс валют

Триггеры: "обнови курс юаня 7.35", "курс доллара 92", "usd_cny 7.4"

```bash
$RM update-currency "usd_cny" 7.35
```
Доступные пары: `usd_cny`, `usd_rub`.

### Обновить стоимость услуг

Триггеры: "страховка 2%", "обрешетка 50", "обнови страховку до 2"

```bash
$RM update-service "insurance_pct" 2
```
Ключи услуг: `insurance_pct` (страховка), `crating_pct` (обрешетка), `palletizing_pct` (паллеты).

### Обновить наценку на категорию

Триггеры: "наценка электроника 1.3", "коэффициент хрупкое 1.5"

```bash
$RM update-surcharge "electronics" 1.3
```

### Показать ставки

Триггеры: "покажи ставки", "текущие тарифы", "ставки", "цены"

```bash
$RM show
```

Форматируй результат:

```
📊 **Ставки FastCargo**

**Гуанчжоу→Москва:**
🚛 Авто: $1.80-3.50/кг (18-25 дн)
🚂 ЖД: $2.30/кг (25-35 дн)
✈️ Авиа: $6.50/кг (5-7 дн)

💰 Валюта: USD | Мин. вес: 30 кг
```

### Показать маршрут

Триггеры: "ставки на маршрут X", "сколько стоит X"

```bash
$RM show-route "Гуанчжоу→Москва"
```

### Обновить ставку (простая)

Триггеры: "обнови авто до 2.90", "поменяй авиа на 7.00"

Когда менеджер просто говорит "обнови авто до X" без указания плотности — обновляй ВСЕ ставки на этот транспорт:

```bash
$RM update-simple-rate "Гуанчжоу→Москва" "авто" 2.90
```

**Маппинг транспорта:**
| Слова менеджера | Ключ |
|---|---|
| авто, автомобиль, фура, машина | auto |
| жд, железка, поезд, железная дорога | rail |
| авиа, самолёт, авиация | air |

### Обновить ставку (по плотности)

Триггеры: "обнови авто для плотности 200 до 2.90"

```bash
$RM update-rate "Гуанчжоу→Москва" "авто" 200 2.90
```

### Добавить маршрут

Триггеры: "добавь маршрут X→Y", "новый маршрут"

```bash
$RM add-route "Иу→СПб" "авто" 3.10 --days-min 20 --days-max 30
```

### Создать файл ставок

Для новой компании:

```bash
$RM init
```

## Правила

1. Эти операции доступны ТОЛЬКО менеджеру
2. После обновления ставки — подтверди: "Ставка авто обновлена: $2.80 → $2.90/кг"
3. Если маршрут не найден — покажи доступные маршруты
4. При добавлении маршрута спроси сроки доставки, если менеджер не указал
